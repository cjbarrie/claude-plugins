# Claude Plugins

A collection of Claude Code plugins.

## Plugins

- **[academic-review](plugins/academic-review/)** — Academic manuscript review tools: peer review manuscripts, parse revision demands, and verify revision responses.

## Installation

Add this marketplace to Claude Code:

```
/plugin marketplace add cjbarrie/claude-plugins
```

Then install a plugin:

```
/plugin install academic-review@cjbarrie/claude-plugins
```
